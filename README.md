### _전역 사용자명/이메일 구성하기_
git config--global user.name "Your name"

# markdown 문법
**markdown** 은 마크업 언어의 일종으로, 존 그루버(John Gruber)와 아론 스워츠(Aaron Swartz)가 만들었다.
읽기쓰기가 쉽다는 장점이 있으나, 멀티미디어 삽입이 불편하고 확장 문법이 많다는 단점이 있다.
확장자는 __.md__ 를 쓴다.
